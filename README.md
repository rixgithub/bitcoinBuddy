## Project Description

BitCoin Buddy is an app utlizing React.js allowing user to set up SMS notification for user defined BitCoin price changes. 

## Technologies Used

* React.js
* Auth0
* HTML/CSS/BootStrap
* CSS
* JavaScript
* Body-parser
* Node.js
* Chart.js
* Express
* MongoDB
* Mongoose
* React-Chart.js
* Cron
* Request
* Twilio
